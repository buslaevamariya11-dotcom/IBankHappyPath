# Тестирование системы авторизации интернет-банка (IBank)

![Java CI with Gradle](https://github.com/buslaevamariya11-dotcom/IBankHappyPath/actions/workflows/gradle.yml/badge.svg)

## Описание проекта

Автоматизация тестирования входа в систему интернет-банка.
Особенность проекта — использование API-запросов для предварительной подготовки (регистрации) данных пользователей перед
выполнением UI-тестов.

## Инструменты

* **Selenide** — для тестирования интерфейса.
* **Rest-assured** — для регистрации пользователей через API.
* **Java Faker** — для генерации тестовых данных.
* **Lombok** — для создания лаконичных data-классов.

## Отчет по времени

* Время, затраченное на ручное тестирование (минут): 20
* Время, затраченное на автоматизацию (минут): 120

## Как запустить тесты локально

1. Запустите SUT в тестовом режиме:
   `java -jar ./artifacts/app-ibank.jar -P:profile=test`
2. Запустите тесты:
   `./gradlew test`
